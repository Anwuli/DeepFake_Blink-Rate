# DeepFake_Blink-Rate


This project detects deepfake videos by analyzing eye blink patterns using OpenFace. Deepfake models often fail to replicate natural blink behavior, making this a reliable biometric cue.

Sample Video: Deepfake vs Real, https://www.youtube.com/watch?app=desktop&v=B4jNttRvbpU

# Technologies

Python, OpenCV, NumPy, Pandas, SciPy

OpenFace
